# 🧪 Demo Web Shop Automation Framework

🔗 GitHub Repo: https://github.com/vinothmca2000/Project_DemoWebShop_Vinoth

---

## 📌 Project Overview

This project is a Hybrid Test Automation Framework developed for testing the Demo Web Shop application. It automates core e-commerce functionalities such as login, product selection, cart operations, and user workflows using Selenium WebDriver.

---

## 🚀 Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Page Object Model (POM)
* Extent Reports

---

## 🏗️ Framework Design

* Hybrid Framework (Data-Driven + POM)
* Modular structure with reusable utilities
* Separation of test logic and page elements

---

## 🔧 Key Features

* Cross-browser testing (Chrome, Firefox, Edge)
* TestNG annotations for execution control
* Data-driven testing using external files
* Extent Reports integration for test reporting
* Reusable utility classes for WebDriver and file handling
* Login & Logout automation workflows

---

## 📂 Project Structure

* `POM` → Page classes (Login, Home, Welcome)
* `Generic Utility` → Reusable methods (File, WebDriver, etc.)
* `Test Scripts` → Test execution classes
* `Reports` → Execution reports

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/vinothmca2000/Project_DemoWebShop_Vinoth
   ```
2. Open in IDE (Eclipse/IntelliJ)
3. Run TestNG XML or test classes
